# Physiological Systems Modelling

This repository contains a collection of computational models, simulations, and analytical studies related to physiological systems.

Developed as part of coursework and independent exploration in biomedical and electronic engineering, the projects focus on understanding biological processes through mathematical modelling and numerical simulation.

---

## 📂 Repository Structure

The repository is organized into separate folders, each representing a specific physiological system or modelling study:

- **Simulation of Respiratory Mechanics**
- **Analysis of Cardiac Physiology**
- **Electrical Properties of Branching Dendrites**
- **Hodgkin-Huxley Model**
- **Muscle/Cartilage Modelling (Paper Presentation)**

Each folder contains its own documentation, code, and results.

---

## ⚙️ Tools & Methods

- MATLAB for numerical simulations
- Mathematical modelling (differential equations, cable theory)
- Matrix-based system solving
- Data visualization and analysis

---

## 🎯 Objectives

- Model physiological systems using engineering principles
- Analyze system behavior under different conditions
- Validate theoretical results with simulations
- Develop intuitive understanding of biological processes

---

## 📌 Notes

Each project includes:
- A detailed report
- Simulation code
- Graphs and results
- Explanations of the modelling approach
